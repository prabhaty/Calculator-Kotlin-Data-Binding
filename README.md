# Calculator-Kotlin-Data-Binding
Simple calculator with kotlin, and data binding.

* Kotlin
* data Binding
* Constraint Layout
* net.objecthunter:exp4j library

# Features: 

* Addition
* Substraction
* Multiplication
* Division
* Decimal number operation
* Percentage
